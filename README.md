# 📊 Paisabazaar Banking Fraud Analysis Dashboard

## 🔍 Project Overview
This project focuses on analyzing banking fraud using a Power BI dashboard. The goal is to identify high-risk customers and uncover fraud patterns using financial and behavioral data.

The dashboard transforms raw data into actionable insights to support better decision-making in financial institutions.

---

## 🧠 Problem Statement
Financial institutions need to:
- Detect fraudulent applications
- Identify high-risk customers
- Monitor credit behavior effectively

This dashboard helps solve these challenges using data visualization.

---

## 🛠️ Tools & Technologies
- Power BI (Dashboard & Visualization)
- Python (EDA - Exploratory Data Analysis)
- Excel / CSV Dataset

---

## 📊 Key Features
- Interactive dashboard with filters:
  - Credit Score Category
  - Age Group
  - Income Group
- KPI metrics:
  - Total Customers
  - Fraud Rate
  - Avg Income & Debt
- Visual analysis:
  - Risk by Occupation
  - Fraud by Age Group
  - EMI vs Credit Score
  - Credit Utilization
  - Payment Delay Analysis
  - Monthly Fraud Trends

---

## 📈 Key Insights
- Fraud risk depends on multiple factors like credit utilization, age, and payment behavior
- Customers aged 36–50 show higher fraud risk
- High credit utilization and delayed payments indicate risky behavior
- Fraud cases increase over time, showing a rising trend

---

## 🎯 Business Impact
- Helps detect fraud early
- Improves decision-making for loan approvals
- Identifies high-risk customers efficiently
